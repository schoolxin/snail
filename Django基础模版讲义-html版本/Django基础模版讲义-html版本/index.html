Introduction

Django基础-模块简介

学习目标

使用 Django 框架来快速开发 Web 应用程序

  1. 如何使用 Django 来创建 Web 项目和 Web 应用
  2. 如何使用 MVT 这种开发模式，model 模型、view 视图、template 模板
  3. 使用 model 来管理和操作数据库
  4. 使用 view 来接受、响应、处理 http 请求
  5. 使用 template 来动态产生 html 页面

学习内容

1、Django开发流程

  • 搭建虚拟开发环境
  • 创建 Django 项目和应用
  • 初步使用模型、视图、模板完成一个 web 页面

2、模型

  • 模型类的定义
  • 数据增删改查操作

3、视图

  • 通过函数创建视图
  • 配置视图的路由
  • 通过 HttpRequest 接受请求数据
  • 通过 HttpResponse 返回响应数据
  • 通过 cookie 和 session 保持会话状态
  • 通过类创建视图

4、模板

  • Django 自带的模板引擎
    • 模板的语法
    • 模板过滤器
    • 模板的继承
  • Jinja2 模板引擎
  • CSRF 跨站点请求伪造的原理和防护
